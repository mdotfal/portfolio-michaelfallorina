Marvel Search API

const skills = [HTML, CSS, JavaScript, jQuery];

This project was created utilizing 3 different API endpoints, 2 of them being from the Marvel API and 1 from the YouTube API. When the user searches for a Marvel SuperHero, they will send a GET request to the Marvel API receiving back a list of Marvel Characters matching the query. As the user navigates and clicks a SuperHero image a modal will display and a GET request to the next Marvel API will return a list of comics that the user will be able to browse. When the user clicks "View Video", the modal will close and a GET request to the YouTube API will return back a list of YouTube API based upon the original search query.
